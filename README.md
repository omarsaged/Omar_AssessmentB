# Dynamic CV Portfolio Website

## Student

Omar Hindawy

## Project Description

This project is a dynamic CV portfolio website created for the Computer Science Ecosystems Assessment B coursework. It rebuilds and improves my Assessment A portfolio website by using dynamic content loaded from a JSON file and JavaScript interaction.

The website includes a homepage and a CV page. The homepage introduces the portfolio, while the CV page displays information about my education, skills, experience, languages, and contact details.

## Files Included

* `index.html`: Homepage of the portfolio website.
* `cv.html`: CV page with dynamic CV sections.
* `style.css`: One shared stylesheet used for all pages.
* `app.js`: JavaScript file that loads data, controls the theme button, show/hide buttons, and scroll animation.
* `data.json`: Stores the CV content used by the website.
* `OmarHindawy_bash.sh`: Bash script that counts new learning comments.
* `OmarHindawy_bash_report.txt`: Output file created by the Bash script.
* `terminal_screenshot.png`: Screenshot of the Bash script terminal output.
* `image for project.png`: Image used on the homepage.

## How To Run Locally

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server extension if needed.
4. Right-click `index.html`.
5. Click `Open with Live Server`.
6. Use the navigation bar to move between the homepage and CV page.
